# My Portfolio Website 🚀

A simple personal portfolio website built with HTML, CSS, and JavaScript.

## About This Project

This is my first web development project! I built this portfolio while learning to code. It's a simple, responsive website that showcases my skills and projects.

## Features

- Responsive design (works on mobile and desktop)
- Smooth scrolling navigation
- Animated skill bars
- Contact form (demo only)
- Mobile hamburger menu

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to View

1. Download or clone this repository
2. Open `index.html` in your web browser
3. That's it!

## What I Learned

- HTML structure and semantic tags
- CSS styling, flexbox, and grid layout
- JavaScript DOM manipulation
- Responsive design principles
- Git version control
